输入

输入格式（每行一个小节，最终会自动拼接为程序需要的 简谱 / 普通话歌词，每个小节用 | 分隔）：
  • 简谱：音符用空格分隔，休止/延音可写 0 / -
  • 高八度：用 ^ 表示高八度，例如 ^1（在谱面中会渲染为数字上方加点）
  • 普通话：建议逐字/短句，系统会按该小节字位数自动对齐（多则截断，少则留空）
  • 小节数：简谱与普通话必须一一对应（行数一致）
每行一个小节：左侧简谱，右侧普通话（会自动拼成用 | 分隔的格式）
主题标签（回车/逗号添加）
